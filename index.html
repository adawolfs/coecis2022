WebAssembly

Un conjunto de instrucciones binarias para una maquina virtual de pila. Fue diseñado para proveer un formato de codigo compilable y portable para otros lenguajes de programacion.

WebAssembly busca brindar entornos de ejecucion rapidos, livianos, portables y seguros.

La maquina de pila de WASM esta diseñada para ser codificada en un formato binario eficinete en tamaño y tiempo de carga.

Tiene como objetivo ejecutar a velocidades nativas, tomando ventaja de las capacidades del hardware disponibles en la mayoria de plataformas.

.WAT

WebAssembly Text File

.WASM

WebAssembly Binary File

Formato binario con una representacion compacta y eficiente en tamaño y tiempo de carga.
